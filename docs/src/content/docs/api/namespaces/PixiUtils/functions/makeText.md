> **makeText**(`pText`, `pFont`, `pSize`, `pColour`): `BitmapText`

Creates and returns a `BitmapText` object.

## Parameters

• **pText**: `string`

The text to display.

• **pFont**: `string`

The font to use.

• **pSize**: `number`

The size of the font.

• **pColour**: `number`= `0x000000`

The colour of the font.

## Returns

`BitmapText`

## Default

```ts
pColour 0x000000
```
